This GitHub repository contains an automation testing project for the ParaBank website. The project is developed using Java, Selenium WebDriver, and TestNG, offering a robust framework for testing online banking applications.

**Project Features**

_**Automated Test Suite:**_ A comprehensive set of test cases covering the key functionalities of the ParaBank website.

_**Selenium WebDriver:**_ Automates user interactions with web elements for testing banking workflows.

_**TestNG Framework:**_ Provides annotations, execution control, and detailed reporting.

_**Page Object Model (POM):**_ Implements a modular and maintainable test structure.

_**Cross-Browser Testing:**_ Ensures consistent application behavior across multiple browsers.

_**Data-Driven Testing:**_ Executes tests with various datasets sourced from external files (e.g., Excel, JSON).

_**Custom Reporting:**_ Generates detailed test execution reports for easy analysis.

**Key Functionalities Tested**

_**User Authentication:**_ Validates registration, login, and logout features.

_**Account Overview:**_ Verifies account summary details, balances, and transaction history.

_**Fund Transfers:**_ Tests the functionality for transferring funds between accounts.

_**Bill Payments:**_ Automates bill payment workflows to ensure accuracy.

_**Request Loan:**_ Validates loan application processes.

_**Navigation:**_ Checks the responsiveness and usability of different web pages.

_**Error Handling:**_ Captures screenshots of failed test cases for debugging purposes.

**Requirements**

_**Java 8+**_
_**Selenium WebDriver**_
_**TestNG**_
_**Maven (for dependency management)**_
